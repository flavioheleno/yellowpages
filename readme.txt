**************************************************************

1. Requirements

To enjoy this application, you'll need:
	- PHP5
	- PDO
	- PDO SQLite Driver
	- Modern webbrowser (Chrome <3)

**************************************************************

2. How to use

Start by running "app/create_db.php", it will create the database
and fill it with some data.
Set the current working directory to the same directory you 
extracted the index.html file and start local server with the 
command: php -S localhost:80.
Point your webbrowser to http://localhost/.
Enjoy the ride.
